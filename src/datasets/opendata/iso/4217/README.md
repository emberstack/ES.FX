# ISO 4217 dataset

The ISO 4217 currency code list bundled in `ES.FX.OpenData.Currencies` (namespace
`ES.FX.OpenData.Currencies.ISO4217`).

## Files

| File | Standard | Contents |
| --- | --- | --- |
| `iso_4217.json` | ISO 4217 | 181 currencies: `alpha_3` (e.g. `USD`), `numeric` (e.g. `840`), `name` (e.g. `US Dollar`). |

The file is JSON (snake_case keys, numeric codes as zero-padded strings) and is embedded (by link) into
`ES.FX.OpenData.Currencies` as an assembly resource (LogicalName `ES.FX.OpenData.Currencies.ISO4217.iso_4217.json`).
